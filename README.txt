A Gmail client application to run on desktop that allows
one to use Gmail account to send email
------------------------------------------------------------------------

PROJECT TITLE: Java Mail Gmail Client

PURPOSE OF PROJECT: Send mail on desktop without needing to open
a web browser 

VERSION or DATE: 1/4/2018

HOW TO START THIS PROJECT: Open project in NetBeans and then run
the project. Input Login information where it says User and Password
that corresponds to an already created Gmail account. Hit 'SEND' when 
finished and if successful an alert box will open saying 'Mail Sent!'
Go to your Gmail settings and allow the use of your email account by 
other applications to avoid problems with Gmail account
